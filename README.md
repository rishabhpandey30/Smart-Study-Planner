##  Project Objective

The objective of this project is to develop a **Smart Study Planner** that allows students to:

- Organize academic subjects  
- Plan daily and weekly study schedules  
- Track assignments and exams  
- Monitor productivity  
- Improve time management  

---

##  Key Functionalities

### Dashboard
- Overview of subjects, tasks, and completion rate  
- Upcoming tasks display  
- Today's schedule  

---

### Subject Management
- Add subjects with priority levels  
- Edit existing subjects  
- Delete subjects  
- Color-coded priority tags  

---

### Schedule Planner
- Create daily & weekly timetables  
- Prevent time overlaps using conflict detection  
- Organized slot display  

---

### Task Manager
- Manage assignments, exams, and study tasks  
- Deadline tracking  
- Overdue task highlighting  
- Completion toggle  

---

### Progress Analytics
- Visual completion rate  
- Productivity insights  

---

### Settings
- Toggle Dark / Light theme  
- Export planner data  
- Reset stored data
